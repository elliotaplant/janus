# Janus

An open source, install once and use forever alternative to Loom.

Outputs .WebM or .mp4 files. No cloud, no subscription, no time limits.

## Packaging

You'll need to create an `ffmpeg-binaries` directory with this structure:

```
ffmpeg-binaries
├── linux
│   └── ffmpeg
├── darwin
│   └── ffmpeg
└── win
    └── ffmpeg.exe
```
